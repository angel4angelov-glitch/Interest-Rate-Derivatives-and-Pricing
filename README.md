# Interest Rate Products – Pricing & Curve Experiments
> ⚠️ Educational sandbox. Not production code or financial advice.

Personal learning repo for IR products: futures, swaps, curve construction, DV01 hedging, convexity adjustments.

## Contents
- `futures/` — SOFR/Eurodollar pricing, margining, P&L
- `swaps/` — vanilla IRS, DV01, hedge ratios
- `curves/` — bootstrapping, interpolation
- ...

## Attribution
Code is adapted from textbooks, courses, and articles. Each file notes its source.

`pip install -r requirements.txt`
